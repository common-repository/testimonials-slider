== Description ==
	This plugin is used to create sliding testimonial. You can add name,image,description,category,Main Box Background Color,Testimonial Section Heading,All Text Color,Client Content Box Color and tag. just use the short code in required section and it will work fine.developed on bootstrap and its mobile responsive.

Features

    Fully Responsive
	1 layouts 
    Slider control
    Widget Supported 
	Testimonial Heading
	Main Box Background Color
	All Text Color
	Client Content Box Color 	


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/Testimonial-Slider` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. You see the admin menu option name "Testimonials".
